## 决策与下一步

v1 揭示**结构性发现**（详见 v1/conclusion.md）：
- VA 机制层完全对称（高/低抛比 0:0），但**在 2020-2024 A股 ETF 上 5 年内 NAV 从未超过 6%-12% 任一档目标路径** → 0 次卖出动作
- 货币池在 2021-08 ~ 2022-03 区间耗尽（取决于 cagr_target），耗尽后 VA 退化为"100% 风险池静态等权"
- **核心 KPI（高/低对称性）兑现**，但绝对收益跑输 S2 v1（VA 没创造新 alpha，只是"以更激进的形式重演 S1 后段"）
- 印证 S2 v2 conclusion 的根因诊断：6 ETF 池在 5 年内整体偏多头是结构性事实——任何"对称做T" 框架在该池子上都会偏向单方向触发

候选 v2 方向（不在 v1 上事后调参）：
- **方向 a**：换池子（5 年回测 BH = 4% CAGR；VA 想做对称必须找年化 ~3% 的标的池子，即"震荡池"——可能是 510300 + 511990 + 161725 白酒等更分散的池子，或单一 510300）
- **方向 b**：动态目标 CAGR（按过去 1 年实际实现 CAGR ± 调整，但有 OOS 不可验证性）
- **方向 c**：放开杠杆 / 注资（让 VA 真正按 Edleson 原版执行；脱离"100k 一次性投入"基线）—— 需要新基线
